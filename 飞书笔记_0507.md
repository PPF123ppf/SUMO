# SUMO-CAV 混合博弈换道模型 — 2026.05.07 进展

## 一、当前模型架构

### 1.1 场景

- 4km 三车道高速，3000m 处中间车道事故封堵
- 混合交通：CAV（渗透率 p%）+ 人类车（1-p%）
- 事故时间线：0-90s 正常 → 90-100s 突发期 → 100s+ 有序疏散期

### 1.2 CAV 决策流程

```
感知（V2X噪声/延迟/丢包）
→ 间距门控 + TTC硬安全门控（突发2.0s / 有序1.5s）
→ 8维原子特征提取
→ 自适应权重（密度/紧迫度 → 安全↑ 速度↓）
→ 混合博弈求解（按后车类型选博弈结构）
→ 渗透率自适应阈值
→ 换道执行（准备0.4s→执行2.8s→稳定0.5s）
→ 在线TD微调
```

## 二、8维原子特征

| 维度 | 特征 | 换道时 | 不换道时 | 含义 |
|:--:|------|--------|---------|------|
| 0 | speed | 目标车道前车速度比 | 自车速度比 | 速度收益 |
| 1 | urgency | 距事故归一化距离 | 0 | 紧迫驱动力 |
| 2 | pressure | 当前车道头距压力 | 0 | 逃离压力 |
| 3 | safe | min(前TTC, 后TTC)评分 | 当前车道前TTC | 安全评分 |
| 4 | coop | 后车协同+制动加分 | 0.05 | 协同奖励 |
| 5 | social | 换道对后车TTC缩减比 | 0 | 社会冲击（创新） |
| 6 | density | 局部车流密度 | 0 | 环境压力 |
| 7 | lc_cost | 1.0 | 0.0 | 换道固有代价 |

> 每个特征独立物理语义，IRL 学出的权重可直接解释。

## 三、混合博弈框架

### 3.1 三种博弈结构 × 三种后车类型

| 后车类型 | 占比 | 博弈结构 | 特征 |
|:--:|:--:|------|------|
| Type 0 自私型 | 20% | **静态 Nash** — 双方同时决策 | 无协同动机 |
| Type 1 合作型 | 60% | **Stackelberg** — 本车Leader，后车最优反应 | CAV基础协同 |
| Type 2 高合作型 | 20% | **Nash合作** — 最大化联合收益 | 强协同，互惠 |

### 3.2 混合策略求解

用 softmax(temperature=0.15) 替代 argmax —— 保留行为随机性，避免决策僵化。

### 3.3 连续后车反应

后车 delta_v 不再固定，根据间距/速度动态计算：
- 间距紧 → 制动强（max -4.5 m/s²）
- 间距宽 → 加速温和（max +2.6 m/s²）

## 四、收益函数

### 本车（8维）

`U_e = w_adapt · f(action, fa) - lc_cost × I(换道)`

自适应权重：w_safe ↑ 随 density/urgency，w_speed ↓ 随 urgency

### 后车（5维 × 3种类型）

`U_f = w_t · [safety, speed_ratio, comfort, coop, gap_safe]`

## 五、低渗透率三机制

| 机制 | 方法 | 效果 |
|------|------|------|
| 对抗性偏差 | 后车=人类 → 加速堵位+10%，减速-10% | 不天真假设人类配合 |
| 互惠记忆 | 记录人类车合作历史 → 动态调先验 | 合作者加分，不合作者警惕 |
| 渗透率自适应阈值 | 10%CAV→0.46×阈值，100%→1.0× | 低渗透下果断，高渗透可挑剔 |

## 六、IRL 训练结果（2026.05.07）

### 数据

AD4CHE（中国拥堵高速，大疆无人机），5段录制，~360万帧，3908个换道片段，69404个8维特征样本。

### 学出权重 vs 手工权重

| 特征 | 手工（有序期） | IRL（学出） | 差异解读 |
|------|:------:|:------:|------|
| speed | 0.30 | **0.15** | 真人换道频率比预期低 |
| urgency | 0.10 | **0.19** | 紧迫度在数据中有真实信号 |
| pressure | 0.06 | **0.18** | 前方贴紧是换道真实驱动力 |
| safe | 0.25 | **0.25** | 安全权重基本一致 ✅ |
| coop | 0.12 | **0.015** | 合作信号轨迹数据难捕捉 |
| social | 0.05 | **0.004** | 同上，需加速度数据 |
| density | 0.05 | **0.11** | 拥堵场景密度有区分度 |
| lc_cost | 1.0 | **0.70** | 真人换道频率高于CAV预期 |

### 结论

- 手工权重偏"效率驱动"（speed=0.30），IRL偏"压力驱动"（urgency+pressure=0.37）
- safe 一致 → 安全偏好是通用的
- lc_cost 差异 → 真人比 CAV 更愿意换道（CAV 太规矩）
- IRL 权重将用于对比实验

## 九、论文叙事线

> 提出面向混合交通的**混合博弈换道决策模型**：8维原子特征提供可解释的决策基础，IRL从真实驾驶数据学习常规场景权重，手工权重补充事故应急场景。通过三种博弈结构（静态Nash/Stackelberg/Nash合作）自适应选择，配合低渗透率下的对抗性偏差、互惠记忆和自适应阈值，解决CAV在混合交通中"冻住"和"博弈失效"问题。

**预测核心发现**：
- CAV渗透率存在效率拐点（预计30-50%）
- 极低渗透率下CAV博弈可能适得其反（10% < 0%）
- IRL权重在高渗透率下是否优于手工权重
